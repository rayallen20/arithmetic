# arithmetic

本仓库用于作业和笔记提交

## 目录说明

### 一级目录

```
./
├── README.md
└── class1
```

`README.md`:仓库说明

`class1 /`:本目录用于存放每课时的代码和笔记.命名风格为:`class + 课时编号`.课时编号从1开始累加计算.

### 二级目录

```
./class1
├── code
└── note

2 directories, 0 files
```

`code/`:本目录用于存放代码

`node/`:本目录用于存放笔记

#### note目录

```
./
```
